# Claude Prompt — CropAnalyser UI/UX Redesign

Redesign the **CropAnalyser** project UI/UX around these goals.

CropAnalyser is a **web platform accessible to anyone**, not only a mobile app.

Target users include:

- Day-to-day farmers
- Agronomists
- Agriculture researchers
- NGOs / institutions
- Corporate / agri-business users

The UI must balance two things:

1. **Simple enough for farmers to use comfortably**
2. **Modern, clean, data-driven, and professional enough for corporate users**

## Core UX Direction

Design philosophy:

**WhatsApp simplicity + Stripe cleanliness + practical agricultural usability.**

Avoid:

- cluttered dashboards
- government-portal style interfaces
- overly technical screens
- excessive text input

Prefer:

- clean cards
- clear navigation
- visual explanations
- large readable UI elements
- responsive layouts
- intuitive workflows

## Platform Requirements

This is a **responsive website**, not phone-only.

Requirements:

- Mobile-friendly
- Desktop-friendly
- Tablet responsive
- Low-friction user flow
- Minimal typing
- Fast navigation
- Accessible design

## UI Structure

Create separate but connected experiences:

### Farmer View
Focus on simplicity.

Main areas:

- Home Dashboard
- Crop Recommendations
- Weather Insights
- Market Prices
- Field Analysis
- Simple Reports

Use:

- easy language
- visual indicators
- confidence scores
- recommendation explanations

### Pro / Corporate View
Focus on analytics and deeper insight.

Include:

- scoring breakdowns
- profitability analysis
- historical trends
- regional intelligence
- model outputs
- export/reporting tools

## CropAnalyser Features To Design Around

Include UI support for:

- Crop recommendation engine
- Soil analysis
- Weather intelligence
- Market/mandi pricing
- Regional crop history
- Profitability scoring
- Smart recommendations
- Future NDVI / satellite analysis support

## Output Required

Generate:

1. Revised product UX strategy
2. Site map / navigation structure
3. Page breakdown
4. Dashboard layout concepts
5. Component system
6. Design system (colors, typography, spacing)
7. Farmer UX mode + Professional UX mode
